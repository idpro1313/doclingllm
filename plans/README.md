# plans/

Сюда режим **Architect** (`mode-architect`) кладёт артефакты планирования:

- `DevelopmentPlan.md` — обязательный план (`devplan-protocol`: Draft Code Graph, data flow, acceptance criteria)
- `business_requirements.md` — по необходимости
- `AppGraph.xml` — при необходимости как отдельный граф (см. `graph-protocol`)

Не начинайте `mode-code`, пока план не согласован с оператором.
