# tests/

Все тесты проекта — **в этой корневой папке** (см. `mode-architect`, `mode-code`).

Для фазы **QA** (`mode-qa`) создайте:

- `tests/test_guide.md` — инструкции для независимой верификации
- `tests/conftest.py` — при использовании Anti-Loop (см. `mode-code`)

Имена файлов: `tests/test_<module>.py`.
